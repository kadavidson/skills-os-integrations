PHASE 01

The org sets the targets

"AI Prototyping is tracking at 67% team readiness — up 45% since October."

L&D admins track skill readiness across the team — not course completions, but verified capabilities. MCP signal shows how the team is actually using AI: prompt length, session frequency, pipeline adoption.

Learner view · My Plan

PHASE 02

The learner sees evidence, not assignments

"Three things have clearly shifted over the past few weeks."

When Marcus opens Coursera, he sees behavioral observations from how he's actually been working — not a self-assessment. Specific evidence from his AI sessions, surfaced via MCP.

The lesson arrives with proof

Each session opens with a specific observation from the learner's own work — a prompt they've actually used and a precise diagnosis of what's missing.

Lab · Apply in the real thing

PHASE 04

No steps until you launch

"Time to apply it — in the real thing."

Two options: jump straight in, or start with a tutorial. The steps only appear as a live floating guide inside the tool — once the session is running, via MCP.

Wallet · Verified skills · Live credential

PHASE 05

Skills as a living record

"5 skills verified — you're ahead of pace."

Not a badge for finishing a course — verified proof grounded in demonstrated behavior. Skills can go stale and need a refresh. The credential link is live: if a skill lapses, it updates automatically.

Why this is the bet

01
Behavioral evidence, not self-report
MCP connects to where learners actually work and surfaces specific observations — making the learning case before the lesson begins.
02
Work-anchored application
The lab happens inside the real tool with a live Coursera guide watching the session and surfacing hints in real time.
03
Org goals connect to individuals
Admin skill targets flow down into each learner's plan automatically — with context about who set the goal, why, and what unlocks next.
04
Verification that means something
Skills are verified through demonstrated behavior — not a quiz score. The wallet is a live credential that updates as skills are earned or lapse.
